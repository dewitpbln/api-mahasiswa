# API Mahasiswa Laravel

Project ini merupakan API server yang dibuat menggunakan Laravel untuk mengelola data Mahasiswa dengan fitur CRUD (Create, Read, Update, Delete).

## Fitur
- GET /api/mahasiswa → menampilkan semua data
- POST /api/mahasiswa → menambahkan data
- GET /api/mahasiswa/{id} → menampilkan detail
- PUT /api/mahasiswa/{id} → update data
- DELETE /api/mahasiswa/{id} → hapus data

## Teknologi
- Laravel
- MySQL
- Postman (Testing)
- Scramble (Dokumentasi API)
